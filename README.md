# Funngro Website Redesign

A modern two-page redesign of the Funngro website created as part of the Funngro evaluation project.

## Features

- Responsive Design
- Modern User Interface
- Dark / Light Mode
- Mobile Friendly
- SEO Optimized
- Smooth Hover Effects
- Clean HTML, CSS & JavaScript

## Pages

- Home
- About

- HTML5
- CSS3
- JavaScript
- Font Awesome

- Lazy Loading Images
